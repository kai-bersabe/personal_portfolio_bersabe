This project demonstrates how JavaScript and Three.js can apply 3D geometric transformations—rotation, translation, scaling, and reflection—on the Spotify logo. It highlights the use of Linear Algebra concepts such as matrices and vectors in real computer graphics development.

Mini-Machine | Work Application System

This Mini-Machine is a rule-based system designed to automate applicant evaluation. It processes experience, scores, portfolio quality, and interview results using decision-based logic. This demonstrates structured programming and problem-solving in real-world workflow automation.
